# Папка для изображений

Разместите ваши физические файлы изображений (JPG, PNG, WEBP) в этой папке.

**ВАЖНО:** Для избежания проблем с кодировкой, пожалуйста, переименуйте ваши файлы, используя **латинские буквы** (английский язык).

### Пример:
Вместо `Апартаменты.jpg` используйте `apartments.jpg`.

### Текущие используемые изображения:
- `apartments.jpg` (Используется в блоке формы захвата "Составим персональную подборку")

### Как использовать в коде:

```tsx
<img src="/assets/images/apartments.jpg" alt="Вилла" />
```
